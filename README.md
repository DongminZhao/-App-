## Introduction

这是一个手机银行app评论分析工具，实现功能主要有：
1. app评论数据爬取
2. 分词与关键词提取
3. 词云与词频统计
4. 情感分析
5. 主题聚类

正常状态下实现效果在: “ 案例：手机银行App评论分析 ”
注意：案例中最后表格需要自己根据LDA结果进行统计
